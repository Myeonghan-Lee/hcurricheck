# 🏫 2027 고등학교 교육과정 점검 도우미

## 🌟 프로젝트 개요

본 시스템은 **2022 개정 교육과정**이 전면 적용되는 **2027학년도 고등학교 교육과정** 편성 지침을 자동 점검하기 위해 개발되었습니다. 엑셀 파일을 직접 분석하여 OCR 오차를 없앴으며, 일반고 지침은 물론 **과학중점학교(서울형)**의 특수 편성 지침까지 완벽하게 지원합니다.

## ✨ 핵심 기능

- **학교 유형별 맞춤 점검:** 일반 고등학교 및 과학중점학교 유형을 선택하여 전용 지침을 적용합니다.
- **엑셀 직접 데이터 추출:** 이미지 OCR 대신 엑셀의 텍스트와 수치를 직접 읽어 분석 정확도를 극대화했습니다.
- **특수 기록 형식 해석:** 엑셀 셀 내에서 두 줄로 표기된 학점(예: `12` 아래 `(택4)`)을 AI가 `12(택4)`로 논리적으로 통합 해석합니다.
- **지능형 위계성 체크:** `한국사1, 2` 등은 제외하고, 로마자(`Ⅰ, Ⅱ`)가 포함된 과목에 대해서만 선후 이수 관계를 점검합니다.
- **보고서 자동 생성:** 점검 결과를 학교별 요약 데이터와 함께 개조식 종합 의견을 포함한 사용자 정의 엑셀 양식으로 내보냅니다.
- **안정적인 분석 로직:** 404 모델 에러 방지(자동 모델 매칭) 및 429 할당량 제한 대응(순차 분석 및 자동 재시도) 기능이 탑재되어 있습니다.

## 📋 주요 점검 항목

### 1. 일반 고등학교 (17개 항목)
- 총 이수 학점(192), 필수 이수 학점(84) 충족 여부
- 국·수·영 총합 81학점 이내 준수 여부
- 체육 10학점 이상 및 매 학기 편성 여부
- 과목 위계성 및 2022 개정 과목 표준 명칭 사용 여부
- 학기 간 이수 학점 격차(5학점 이내) 점검 등

### 2. 과학중점학교 (추가 5개 항목)
- 1학년 과학 4과목(통합과학1, 2, 과탐실1, 2) 10학점 필수 편성
- 과학 8과목(물/화/생/지 Ⅰ·Ⅱ) 및 관련 진로 선택 과목 다양성
- 수학/정보 교과 심화 과목(미적분Ⅱ, 기하, AI 기초 등) 필수 편성
- 융합 선택 과목(과학의 역사와 문화 등) 편성 권장 준수 여부

## 📄 보고서 출력 양식
내보내기 기능을 통해 생성되는 엑셀 파일은 아래와 같은 양식을 따릅니다:
1. **첫 번째 줄:** 학교명 표시
2. **두 번째 줄:** 항목명 | 판정 | 상세근거 헤더
3. **데이터 줄:** 17~22개 항목별 상세 분석 데이터
4. **마지막 줄:** 개조식으로 작성된 종합 개선 사항(의견)
